NoSmoke

A cross platform UI crawler which scans view trees then generate and execute UI test cases.

Quick Start

Cross Platform

Supports android and iOS, setup and run at anytime!

Automated UI Tests

Simpler than UI BDD scripts, more accurate than monkey tests.

Report Generation

Testing result is fully visualized via macaca-reporter

# Getting Started

# install
$ npm i nosmoke -g

COMPATIBILITY NOTE

NoSmoke requires Node.js >= 8.
